# Cronograma V7.5 con Drag & Drop

Cronograma inteligente con funcionalidad de drag & drop y recalculación automática de horarios.

## Características

- ✅ Integración con Todoist
- ✅ Sincronización con iCloud Calendar
- ✅ Integración con Firefly III
- ✅ Widget de BOE Subastas
- ✅ Estadísticas de Idealista
- ✅ **Drag & Drop** para reorganizar tareas
- ✅ **Recalculación automática** de rangos horarios
- ✅ Enlaces directos a Todoist en cada tarea
- ✅ Checkboxes funcionales

## Despliegue en Railway

1. Crear proyecto nuevo en Railway
2. Añadir PostgreSQL
3. Configurar variables de entorno
4. Desplegar desde este repositorio
